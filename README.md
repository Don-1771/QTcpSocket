# TCP/SSL Client (Qt C++)

## Описание проекта

Данный проект представляет собой клиентское приложение на C++ с использованием Qt, реализующее подключение к удалённому серверу по TCP с поддержкой SSL (HTTPS).

Клиент устанавливает защищённое соединение, отправляет HTTP-запрос и выводит полученные данные в консоль.
Этот проект предназначен для изучения:

работы с сетевыми сокетами
HTTPS соединений
Qt event-driven архитектуры
SSL/TLS handshake процесса

---

## Цель проекта

Цель работы — изучение:
- клиент-серверного взаимодействия через TCP
- работы с SSL/TLS в Qt
- класса `QSslSocket`
- обработки сетевых событий через signals/slots
- базового HTTP-запроса через сокеты

---

## Используемые технологии

- C++17
- Qt Network
  - QSslSocket
  - QTcpSocket (альтернативно)
  - QNetworkProxy
- Qt Core
- QCoreApplication

---

## Архитектура

Основной компонент проекта — класс `Client`, который управляет:
- установлением соединения
- SSL-рукопожатием
- отправкой HTTP-запроса
- обработкой ответов сервера
- обработкой сетевых ошибок и состояния соединения
